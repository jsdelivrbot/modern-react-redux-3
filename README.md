# React-Redux: Course by Stephen Grider

### Getting Started

Checkout this repo, install dependencies, then start the process with the following:

```
> git clone https://github.com/kpunith8/modern-react-redux-.git
> cd modern-react-redux
> npm install
> npm start
```
